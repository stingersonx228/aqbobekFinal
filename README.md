# 🛠 Инструкция по интеграции модуля Aqbobek AI

Этот репозиторий содержит готовый модуль обмена сообщениями и ИИ-аналитики для школьного портала.

## 📁 Структура модуля
- `main.py` — Главный API сервер (FastAPI). Содержит эндпоинты для фронтенда и вебхуки для ботов.
- `bot.py` — Асинхронный мост для Telegram.
- `src/ai_service.py` — "Мозги" системы (Groq Llama 3.3). Парсит сообщения, определяет роли и важность.
- `src/models.py` — Схема базы данных (SQLAlchemy).
- `src/scheduler_service.py` — Логика умных замен учителей.

## 🚀 Как запустить (для разработчиков)

1. **Окружение:**
   ```bash
   pip install -r requirements.txt
   ```
2. **Настройка:**
   Скопируйте `.env.example` в `.env` и вставьте свои ключи.
3. **Запуск Бэкенда:**
   ```bash
   uvicorn main:app --port 8001
   ```
4. **Запуск Telegram-моста:**
   ```bash
   python bot.py
   ```

## 🔌 API для Фронтенда

### 1. Сводка чата (Chat Summary)
- **Endpoint:** `GET /api/v1/messages`
- **Описание:** Возвращает историю сообщений из всех мессенджеров с ролями и флагом важности.

### 2. Реальное время (WebSockets)
- **URL:** `ws://localhost:8001/ws`
- **Событие:** `NEW_MESSAGE`
- **JSON Format:**
  ```json
  {
    "type": "NEW_MESSAGE",
    "data": {
      "sender": "Имя",
      "role": "Учитель истории",
      "content": "Текст сообщения",
      "isImportant": true
    }
  }
  ```

### 3. Умные замены
- **Endpoint:** `GET /api/v1/schedule/substitution?teacher=Имя`

## 🤖 Логика ИИ
ИИ автоматически обогащает каждое сообщение:
- Определяет **роль** отправителя.
- Ставит флаг **важности**.
- Классифицирует тип (IT-поддержка, Логистика, Столовая, ЧП).
