# 🚀 Aqbobek Lyceum AI OS: Killer Features

Этот документ содержит описание ключевых функций системы, которые выделяют наш проект на хакатоне.

## 1. 🧠 Smart Substitutions (Умные замены)
Система не просто фиксирует отсутствие учителя, а **автоматически анализирует расписание** (извлеченное из PDF) и подбирает замену.
*   **Как работает:** Ищет свободных учителей с таким же профилем. Если их нет — ищет любого свободного дежурного учителя.
*   **Команда:** `/free [время]` (например, `/free 09:05-09:50`) — покажет всех свободных учителей в этот слот.

## 2. 🚨 Multi-Platform Emergency Broadcast
При обнаружении экстренной ситуации (драка, пожар, мед. помощь), ИИ мгновенно повышает приоритет до `CRITICAL`.
*   **Killer Feature:** Система одновременно рассылает уведомления во ВСЕ подключенные мессенджеры (WhatsApp и Telegram) охране и администрации, а также выводит алерт на дашборд через WebSockets.

## 3. ⚙️ AI Service Routing (Маршрутизация заявок)
ИИ понимает контекст сообщения и автоматически направляет его нужному исполнителю:
*   **IT-Support:** Проблемы с техникой.
*   **Maintenance:** Поломки мебели/здания.
*   **Logistics:** Запросы на снабжение (вода, бумага).
*   **Canteen:** Посещаемость для столовой.

## 4. 📊 On-Demand Analytics & Reporting
Директор может в любой момент выгрузить полный Excel-отчет прямо в чат мессенджера.
*   **Команда:** `/report` — генерация и отправка файла в реальном времени.

## 5. 🔌 Unified Messaging Bridge
Полная синхронизация WhatsApp и Telegram. Неважно, где пишет учитель — ИИ обрабатывает всё в едином центре управления, сохраняя историю в общую базу данных SQLAlchemy.
