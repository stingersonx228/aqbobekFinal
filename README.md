# 🚀 Aqbobek AI Orchestrator: The School OS

**Aqbobek AI Orchestrator** — это не просто чат-бот, это полноценная "операционная система" для управления лицеем Aqbobek, созданная для хакатона 2025. Система переводит хаотичные сообщения в WhatsApp в структурированную SQL-базу данных и автоматизирует 90% школьной рутины.

---

## 🔥 KILLER FEATURES (Для Демонстрации)

### 🎙 1. Voice-to-Action (Whisper-v3-Turbo)
Забудьте про печатание. Завуч записывает голосовое сообщение: *"В 205 кабинете сломался монитор, и еще подготовьте приказ о поощрении 7А класса"*.
*   **AI результат:** Система автоматически создает тикет для IT-специалиста и генерирует черновик бюрократического приказа.

### 🧩 2. Smart Substitution (Алгоритм "Домино")
Когда учитель заболевает, бот не просто ищет замену. Он:
*   Анализирует **предметную компетенцию** (SQL Teacher Load).
*   Проверяет **окна** в расписании.
*   Мгновенно отправляет **Push-уведомление** замещающему учителю в WhatsApp.

### 📊 3. Автопилот Столовой (Canteen RAG)
Бот собирает отчеты от учителей ("В 7Б сегодня 20 детей, двое болеют") и ровно в 09:00 выдает завпроизводству столовой готовый отчет в Excel: *"Всего 380 порций. Динамика посещаемости приложена"*.

### 🌀 4. Smart Lenta (Singapore Methodology)
Автоматическое распределение детей по уровневым группам. Алгоритм за 1.2 секунды бронирует кабинеты и назначает учителей на параллельные уроки, исключая конфликты в расписании.

---

## 🛠 ТЕХНОЛОГИЧЕСКИЙ СТЭК (Senior Level)

*   **Brain:** Groq AI (Llama-3.3-70b-versatile) — для извлечения смыслов.
*   **Ear:** Whisper-large-v3-turbo — для транскрибации голоса.
*   **Database:** SQLAlchemy + SQLite (Relational ERP Model). Полная нормализация (Учителя, Кабинеты, Нагрузка, Расписание).
*   **Backend:** FastAPI (Python 3.12) + Node.js (WhatsApp Bridge).
*   **Security:** X-Internal-Token аутентификация и защита от дублей (Message Deduplication).

---

## 🏗 АРХИТЕКТУРА ДАННЫХ

Система мигрировала с плоских JSON-файлов на **полноценную реляционную модель**:
- **Teachers <-> Subjects**: Связи "многие-ко-многим" по нагрузке.
- **Schedule Engine**: Поиск окон и свободных кабинетов через SQL-запросы.
- **Export Engine**: Динамическая генерация Excel-отчетов на основе живых данных.

---

## 🚀 КАК ЗАПУСТИТЬ (Quick Start)

1.  **Backend:** `uvicorn main:app --port 8001`
2.  **WhatsApp Bridge:** `cd wa-service && node index.js`
3.  **Импорт данных:** `python scratch/import_excel_to_sql.py` (загружает нагрузку из Excel).
